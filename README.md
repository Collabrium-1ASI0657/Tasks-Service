# Tasks-Service
Microservice of the tasks bounded context

## Description
This is a Spring Boot microservice for managing tasks.

## Prerequisites
- Java 17 or higher
- Maven 3.9.11 or higher (or use the provided Maven wrapper)
- PostgreSQL database

## Building the Project
```bash
./mvnw clean install
```

## Running the Application
```bash
./mvnw spring-boot:run
```

## Project Structure
- `src/main/java/tasks/microservice` - Main application code
- `src/main/resources` - Configuration files
- `src/test/java/tasks/microservice` - Test code
